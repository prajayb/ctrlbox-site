# CtrlBox Static Website (Starter)

## Quick start
- Open `index.html` in a browser.
- Replace:
  - `assets/media/intro.mp4 (replaced with CtrlBoxIntro.mp4)` with your intro video
  - `assets/media/intro-poster.jpg` with a poster image
  - `assets/img/og-cover.jpg` with your OpenGraph cover image

## Deploy (GitHub Pages)
1. Create a GitHub repo and upload these files.
2. Repo Settings → Pages → Deploy from branch → main → /root
